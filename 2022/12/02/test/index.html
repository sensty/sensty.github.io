# VHDL 的一些技巧

  • 如果要使用 rising_edge 则需要添加 use ieee.std_logic_1164.all （ use ieee.std_logic_1164.std_ulogic 和 use ieee.std_logic_1164.rising_edge ）

  • 在同步语句中，异步执行语句放在同步语句前边

  • signal 相当于网表
    如果用到了门电路，综合就会产生 LUT

  • 循环语句主要是起到简化代码书写和增加代码可读性的作用，此时，循环语句中控制循环次数的变量不会跟元件中的任何输入、输出端口或者信号量或者变量进行运算或比较，所以无论其它信号怎么变化都不会影响循环实现的功能

  • 循环语句的循环次数必须是有限的，因为循环语句是串行的，它必须依附于进程、过程或者函数语句，而进程本身就是一个不断循环的语句体，那么如果一个循环内部套了一个无限循环，那么显然，从代码上面我们就无法解释我们描述的到底是一个什么东西，并且从硬件上也得不到有意义的解。

  • 循环语句通常不应该作为功能语句使用，因为所有的循环功能都可以通过进程的循环特性来描述和表示。而循环语句作为功能描述时的抽象层级过高，很难确定编译器到底会怎么理解。
